Please feel free to choose a distribution most appropriate for you.

Binary Distribution

This a ready to run distribution of PerfCake. All you need is an installation of Java SE 8 or later (look for JDK).

This distribution format is the latest released version. It is based on the master branch in the source code repository on GitHub.

The binaries can be run on Linux, MacOS X and Windows platforms with Java SE 8 installed, both x86 and x86_64 architectures.
